# Pimcore Schema
Schema.org type builder and ld+json generator for pimcore. 
This bundle requires the `spatie/schema-org` package.

* Pimcore >= 6.0.0

## Installation

```json
"require" : {
    "dachcom-digital/schema" : "~1.1.0",
}
```
